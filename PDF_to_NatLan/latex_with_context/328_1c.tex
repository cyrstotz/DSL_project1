**Entropy Balance**  
For a closed system, the entropy balance is:  
\[
\Delta S = \Sigma \frac{Q}{T} + \dot{S}_{\text{gen}}
\]  
The entropy production rate is expressed as:  
\[
\dot{S}_{\text{gen}} = \dot{Q}_{\text{out}} \left( \frac{1}{T_R} - \frac{1}{T} \right)
\]
