The entropy balance equation is given as:  
\[
0 = \dot{m} \left( s_c - s_a \right) + \frac{\dot{Q}}{T} + \dot{S}_{\text{gen}}
\]  

Rearranging for entropy generation:  
\[
\dot{S}_{\text{gen}} = \dot{m} \left( s_a - s_c \right) - \frac{\dot{Q}}{T}
\]
