\[
0 = \dot{m} \left( s_0 - s_6 \right) + \frac{Q}{T_0} + s_{\text{env}}
\]  
Expanding the entropy terms:  
\[
s_{\text{env}} = \dot{m} \left( s_6 - s_0 \right)
\]  
\[
s_{\text{env}} = \dot{m} \cdot c_p \ln \left( \frac{T_6}{T_0} \right) - R \ln \left( \frac{p_2}{p_1} \right)
