\[
\epsilon_K = \frac{Q_{\text{in}}}{W} = \frac{\dot{Q}_K}{\dot{W}}
\]

The heat transfer \( Q \) is expressed as:  
\[
Q = \dot{m} (h_1 - h_2) + Q
\]

Rearranging gives:  
\[
Q = \dot{m} (h_2 - h_1)
\]

\[
h_1 = h_f + x (h_s - h_f)
