\[

\]  
The enthalpy values are:  
\[
h_e = h_f \, \text{at} \, T_{\text{in,12}} = 20^\circ\text{C} \, \text{(Table A-2)} = 2538 \, \frac{\text{J}}{\text{kg}}

\[
h_a = h_f \, \text{at} \, T_{\text{Reactor,1}} = 100^\circ\text{C} \, \text{(Table A-2)} = 2676.1 \, \frac{\text{J}}{\text{kg}}
\]

\[

\]
